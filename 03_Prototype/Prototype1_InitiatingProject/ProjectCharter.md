# Project Charter - Pertemuan 1
**Nama Proyek:** Project Manager Lite (Aplikasi Android)  
**Pemilik:** Hafizh Hilman Asyhari  
**Deskripsi Singkat:** Aplikasi Android sederhana untuk menyimpan dan menampilkan daftar proyek mini (judul, deskripsi, owner, tanggal mulai).  
**Tujuan:** Praktik dasar pengembangan aplikasi Android (Kotlin), memperkenalkan dokumen inisiasi proyek (charter) dan deliverable awal.  
**Deliverables:**
- Aplikasi Android yang dapat menampilkan daftar proyek (RecyclerView).
- Fitur menambah proyek baru (Dialog sederhana).
- Halaman detail proyek.
**Stakeholders:** Dosen mata kuliah, mahasiswa (pengguna), tim pengembangan (kamu).  
**Waktu Pelaksanaan:** 1 minggu (Pertemuan 1).  
**Keterbatasan / Asumsi:** Offline, menyimpan data hanya di memory (untuk versi awal).
